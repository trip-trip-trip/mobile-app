# gallery/ — 보관함

## 기능 요구사항

### 목록 화면
- 진행중인 여행 / 지난 여행 섹션 구분
- 각 여행을 폴더 형태로 표시 (제목, 장소, 날짜, 썸네일)
- 진행중 여행 폴더에서 "친구 초대하기" 버튼 → invite 화면으로 이동

### 상세 화면
- 여행 세부 정보 (제목, 장소, 기간, 참여자)
- 해당 여행에서 촬영한 사진/영상 그리드 또는 타임라인 표시
- 지난 여행: 현상 완료된 미디어 열람 (필터 효과 적용 상태)
- 진행중 여행: 미리보기 불가 (필름 현상 전 컨셉)

## 화면 구성

- `index.tsx` — 여행 폴더 목록
- 추가 화면이 필요할 경우 `[id].tsx` 등 동적 라우트 활용

## 구현 참고

- 목록 데이터: React Query의 useQuery + 무한스크롤 (필요 시)
- 이미지 표시: expo-image (캐싱, 플레이스홀더 지원)
- 영상 재생: expo-av
