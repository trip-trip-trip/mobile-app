# invite/ — 친구 초대

## 진입 경로

gallery → 진행중 여행 선택 → "친구 초대하기" 버튼 → 이 화면

> 독립 탭이 아님. 탭바에 표시하지 않으며, 스택 네비게이션으로 push.

## 기능 요구사항

- 현재 여행에 참여중인 일행 프로필 목록 표시
- 카카오톡으로 초대 링크 공유 버튼
- 초대 링크 클릭 시 앱 설치 → 해당 여행 합류 (딥링크)

## 구현 참고

- 초대 링크 생성: expo-linking을 활용한 딥링크/유니버설 링크
- 카카오톡 공유: react-native-kakao-share 또는 Share API 활용
- 참여자 목록: React Query로 서버 데이터 조회
