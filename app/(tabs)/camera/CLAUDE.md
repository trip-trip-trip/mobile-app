# camera/ — 촬영

## 기능 요구사항

### 사진
- 하루 최대 **4장** 촬영 제한
- 촬영 잔여 횟수 UI 표시 필요
- 제한 초과 시 촬영 버튼 비활성화 + 안내

### 영상
- 촬영 길이 **3초 고정** (자동 종료)
- 촬영 횟수 **무제한**
- 녹화 진행 표시 (3초 프로그레스)

### 공통
- 사진/영상 모드 전환 UI
- 필름 카메라 스타일 UI + 시각 필터 효과
- 촬영 결과는 현재 진행중인 여행에 자동 연결

## 화면 구성

- `index.tsx` — 카메라 뷰파인더 + 촬영 컨트롤

## 구현 참고

- expo-camera 사용
- 필터 효과: GL shader 또는 expo-gl / skia 검토
- 촬영 제한 로직은 서버 검증 + 클라이언트 캐시 병행
- 미디어 저장: expo-media-library 또는 서버 업로드
